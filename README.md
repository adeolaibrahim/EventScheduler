# Event Scheduler (C++)

This project demonstrates basic object-oriented programming in C++ with three classes:
- **Time**: Represents hours and minutes with validation.
- **Date**: Represents year, month, and day with validation.
- **Event**: Represents an event with title, date, start time, and end time.
